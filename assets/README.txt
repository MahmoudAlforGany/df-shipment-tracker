Drop the following two image files into THIS folder:

  1. delta-logo.png      (recommended 256x256 transparent PNG)
  2. mahmoud-photo.png   (recommended 256x256 square portrait)

The app picks them up automatically. If either file is missing, a
graceful "DF" / "MA" initials badge is shown instead of a broken
image icon — so the UI never looks broken.

To change the file paths, edit the two <img src="assets/..."> lines
near the top of index.html.
